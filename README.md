# react-node-dev-connector
A social network web app built with TypeScript, Node.js, Express, MongoDB, React Context API, React Hooks &amp; Styled-components.

## 主要技术（Tech）
### 前端开发（Frontend）
* React + React Router + React Hooks + React Context

* TypeScript

* Styled Components

* Styled Icons

### 后端开发（Backend）
* Node.js
