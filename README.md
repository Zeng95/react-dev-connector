# react-dev-connector
A full-stack social network web app built with TypeScript, React Router, React Context API, React Hooks, Styled-components, Node.js, Express & MongoDB

## 主要技术（Tech）
### 前端开发（Frontend）
* React + React Router + React Hooks + React Context

* TypeScript

* Styled Components

* Styled Icons

### 后端开发（Backend）
* Node.js
